# Secure Application Design

Entrega parcial (mitad del proyecto): backend Spring Boot.

## Contenido incluido en esta mitad

- API REST con Spring Boot
- Registro y login
- Hash de contraseñas con BCrypt
- Endpoint protegido con token Bearer
- Configuracion de seguridad (CORS + filtros)

## Ejecutar backend local

```bash
cd spring-server
mvn spring-boot:run
```

Servidor local:

- http://localhost:8443

La siguiente mitad incluira cliente Apache (HTML+JS async) y documentacion de despliegue.
